# 开发环境建议

## 推荐 IDE

如果你问我的单一推荐，我建议你用：

## `Visual Studio Code`

原因：

- 对 `Tauri + Rust + React` 组合支持最好
- 插件生态成熟
- Windows 和 Linux 体验一致
- 调试前端、Rust、Markdown、SQLite 都比较顺手
- 项目还在快速演进阶段，VS Code 的轻量和灵活度更合适

## 推荐插件

- `rust-analyzer`
- `Tauri`
- `ESLint`
- `Prettier`
- `Tailwind CSS IntelliSense` 如果前端使用 Tailwind
- `Even Better TOML`
- `Markdown All in One`
- `Error Lens`

## 可替代方案

### `RustRover`

优点：

- Rust 体验更强
- 大型 Rust 项目导航能力很好

缺点：

- 前端体验不如 VS Code 灵活
- 对多技术栈小团队项目不一定更省事

### `WebStorm + Rust 插件`

优点：

- 前端体验强

缺点：

- Rust 一体化体验不如 VS Code + rust-analyzer 直接

## 结论

如果只能选一个 IDE：

- 选 `Visual Studio Code`

如果后期 Rust 规模变大，再考虑给核心开发切 `RustRover`。
